# 🚀 快速部署流程图

```

└────────────────────────┬────────────────────────────────────────┘
                         │
                         │ 复制代码
                         ↓
┌─────────────────────────────────────────────────────────────────┐
│  步骤 1: 创建新 GitHub 仓库                                      │
│  ─────────────────────────────────────────                      │
│  □ 创建新仓库: your-company-website                             │
│  □ 复制代码到新目录                                              │
│  □ 修改公司信息:                                                 │
│    • package.json (名称、版本)                                  │

│    • src/data/products.ts (产品信息)                            │
│    • src/app/page.tsx (首页内容)                                │

                         │

┌─────────────────────────────────────────────────────────────────┐

│    • Build Command: npm run build                               │

│  □ 点击 "Deploy" 按钮                                            │
│  □ 等待部署完成 (2-5 分钟)                                       │

│  □ 添加/修改 DNS 记录:                                           │
│    ┌──────────────────────────────────────────┐                │
│    │ Type: A                                  │                │
│    │ Host: @                                  │                │
│    │ Points to: 76.76.21.21                   │                │
│    │ TTL: 3600                                │                │
│    └──────────────────────────────────────────┘                │
│    ┌──────────────────────────────────────────┐                │
│    │ Type: CNAME                              │                │

│    │ TTL: 3600                                │                │
```

## 🔄 日常维护流程

```bash
# 1. 修改代码
vim src/app/page.tsx

# 2. 本地测试
npm run dev
# 访问 http://localhost:3000 测试

# 3. 提交更改
git add .
git commit -m "Update homepage content"

# 4. 推送到 GitHub
git push origin main

# 5. Vercel 自动部署
# 无需额外操作！
# 1-2 分钟后自动更新生产网站
```

## 🆘 快速问题排查

| 问题 | 解决方案 |
|------|---------|
| 域名无法访问 | 检查 DNS 是否正确配置，等待 DNS 传播 |
| 构建失败 | 检查代码是否有 `export const dynamic = 'force-dynamic'` |
| HTTPS 不工作 | Vercel 自动配置需要 5-10 分钟，等待即可 |
| 更新不生效 | 确认 Git 推送成功，检查 Vercel Dashboard 部署状态 |
| WIX 邮箱不工作 | 确保保留了 MX 记录 |
